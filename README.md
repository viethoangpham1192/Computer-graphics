# Computer-graphics

Mô tả dự án:
- Đây là sản phẩm của một dự án môn đồ họa máy tính. 
- Đây là ứng dụng web mô phỏng 3D phòng truyền thống Đại học Quốc gia Hà Nội.

Số lượng thành viên:
- 3

Vai trò trong dự án:
- Lập trình 
- Lên ý tưởng 
- Tối ưu code 
- Sửa lỗi 
- Thiết kế đồ họa

Công nghệ sử dụng:
- JavaScript
- ThreeJs
- Blender
